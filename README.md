# VUTTR

## Install Dependencies

```bash
$ yarn || npm install
```

## Run Storybook

```bash
$ yarn storybook || npm run storybook
```

## Run App

```bash
$ yarn start || npm start
```

## Stack

Production

- React
- Rematch(Redux)
- Redux-Logic
- Emotion
- Formik
- axios

Development & Testing

- Storybook
- Jest
- testing-library

Build
